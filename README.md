# 🧮 Calculadora Web Simples

Este projeto é uma calculadora funcional desenvolvida como parte das atividades práticas iniciais do curso. A aplicação executa as quatro operações matemáticas básicas utilizando tecnologias fundamentais do desenvolvimento web.

## 🛠️ Tecnologias Utilizadas

O projeto foi construído de forma integrada em um único arquivo estruturado, aplicando os seguintes conceitos:

*   **HTML5:** Estruturação dos elementos de interface (botões de comando, teclado numérico e tela de exibição).
*   **CSS3:** Layout responsivo construído com **CSS Grid** para o alinhamento do teclado e estilização visual moderna (modo escuro com destaques nos operadores).
*   **JavaScript (ES6):** Lógica matemática de computação dos dados, captura de eventos de clique do usuário (`onclick`), manipulação dinâmica do DOM e tratamento de erros (como divisões por zero ou expressões inválidas).

## 📊 Funcionalidades

A calculadora executa as seguintes operações:
*   ➕ **Soma** (Operador `+`)
*   ➖ **Subtração** (Operador `-`)
*   ✖️ **Multiplicação** (Operador `*`)
*   ➗ **Divisão** (Operador `/`)
*   🧹 **Limpeza de memória** (Botão `C`)

## 💻 Como executar localmente

Caso queira rodar o projeto no seu computador:
1. Faça o clone deste repositório ou baixe o código fonte.
2. Dê um duplo clique no arquivo `index.html`.
3. O projeto abrirá instantaneamente em qualquer navegador moderno, sem a necessidade de instalar dependências.
